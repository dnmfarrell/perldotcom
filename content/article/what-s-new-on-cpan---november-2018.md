Welcome to "What's new on CPAN", a curated look at last month's new CPAN uploads for your reading and programming pleasure. Enjoy!

APIs & Apps
-----------
* Get a perly interface to Cisco Firepower Management Center using [Net::Cisco::FMC::v1]({{< mcpan "Net::Cisco::FMC::v1" >}})
* [Mail::DataFeed::Abusix]({{< mcpan "Mail::DataFeed::Abusix" >}}) can send SMTP transaction data to the Abusix transaction feed
* [Neo4j::Driver]({{< mcpan "Neo4j::Driver" >}}) is an unofficial implementation of the Neo4j Driver API
* [REST::Client::CrossRef]({{< mcpan "REST::Client::CrossRef" >}}) provides a client for the CrossRef API


Config & Devops
---------------
* Parse a distribution name using [Parse::Distname]({{< mcpan "Parse::Distname" >}})


Data
----
* [DBIx::Class::Smooth]({{< mcpan "DBIx::Class::Smooth" >}}) provides more sugar for DBIx::Class
* Bools aren't just for JSON! Use them everywhere with [Data::Bool]({{< mcpan "Data::Bool" >}})
* [Data::Dict]({{< mcpan "Data::Dict" >}}) supplies a hash-based dictionary object
* Parse the tabular data format generated by database SELECTs with [Table::BoxFormat]({{< mcpan "Table::BoxFormat" >}})
* [Graphics::Skullplot]({{< mcpan "Graphics::Skullplot" >}}) can plot the result of a SQL select query


Development & Version Control
-----------------------------
* [Mojo::AsyncAwait]({{< mcpan "Mojo::AsyncAwait" >}}) is an Async/Await implementation for the Mojolicious ecosystem
* [Path::Tiny::Glob]({{< mcpan "Path::Tiny::Glob" >}}) is a file globbing utility which supports `**`
* Enforce using the /a or /aa modifier with regular expressions with [Perl::Critic::Policy::RegularExpressions::RequireDefault]({{< mcpan "Perl::Critic::Policy::RegularExpressions::RequireDefault" >}})
* Don't permit reuse of names for different types of variables using [Perl::Critic::Policy::Variables::NameReuse]({{< mcpan "Perl::Critic::Policy::Variables::NameReuse" >}})


Hardware
--------
* [Device::Chip::ADT7470]({{< mcpan "Device::Chip::ADT7470" >}}) is a chip driver for the ADT7470


Web
---
* Parse the HTTP header 'Accept-Charset' using [HTTP::AcceptCharset]({{< mcpan "HTTP::AcceptCharset" >}})
* Validate mails against DMARC policy asynchronously using [Mail::DMARC::Iterator]({{< mcpan "Mail::DMARC::Iterator" >}})
* Export a Mojolicious website to static files using [Mojolicious::Command::export]({{< mcpan "Mojolicious::Command::export" >}})
* [Plack::Middleware::Debug::RefCounts]({{< mcpan "Plack::Middleware::Debug::RefCounts" >}}) debug Plack apps with refcounts
* Super-convenient blogging with Markdown and Dropbox: [Plerd]({{< mcpan "Plerd" >}})
